PaleoAnth SiteMaps

About

This repository and website serve as a means to share, create, and visualize spatiotemporal information related to hominin sites in the middle and late Pleistocene. The information below includes both an interactive map and a dynamic table. Tables and maps are able to be exported into a word processor for manuscript development.

m <- leaflet(data = coords)|> addTiles() |>
addProviderTiles(providers$Esri.WorldImagery) |>
  addMarkers(~Lng, ~Lat, 
  popup = paste("Site:", coords$Site, "<br>",
                "Age:", coords$Age),
  icon = my_local_icon_path,
  label = ~Number,
  labelOptions = labelOptions(noHide = TRUE, textOnly = TRUE, direction = 'left',

    style = list(
